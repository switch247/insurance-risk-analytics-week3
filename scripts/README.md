# Scripts

This directory contains executable scripts and legacy helper modules.

## Contents

- `data_loader.py`: Legacy data loading functions (moved to `src/data_loader.py`).
- `plot_util.py`: Legacy plotting utilities (moved to `src/plot_util.py`).
- `preprocess.py`: Legacy preprocessing functions (refactored into `src/preprocess.py`).
- `utils.py`: Legacy utility functions.
- `model.py`: Model definition scripts.
- `predict.py`: Prediction scripts.

**Note:** For the main analysis notebooks, please use the modules in `src/`.
